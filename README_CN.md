# Bakubaku 🏴‍☠️

**AI 驱动的智能磁盘清理工具**

> 取名自海贼王吞吞果实（Baku Baku no Mi）— 吃掉磁盘上的垃圾。

Bakubaku 是一款超越传统清理工具的智能磁盘管家。它不只告诉你什么占了空间，更能**理解**你的文件 — 用大白话告诉你每个文件是什么、能不能删、为什么。

## 核心功能

- **秒级扫描** — 利用 NTFS MFT 直读技术实现极速全盘扫描
- **AI 人话解释** — 大模型用通俗语言解释每个文件（"这是微信的图片缓存，删掉不影响聊天记录"）
- **三层智能分类** — 规则引擎（90%）→ 启发式分析（5%）→ LLM 分析（5%）
- **卡片式清理** — 按场景分卡片展示（"应用缓存 — 3.2 GB"），不用面对密密麻麻的文件列表
- **智能查重** — 识别相似照片、多版本文档（报告v1、报告_final、报告_final_final...）
- **残留清理** — 检测已卸载软件的遗留文件，告诉你它们属于哪个软件
- **可撤销清理** — 文件先进隔离区，7 天内可恢复，不怕删错
- **记忆系统** — 学习你的磁盘布局和清理习惯，越用越快越准
- **隐私优先** — 所有分析在本地完成，文件内容绝不上传
- **双界面** — 普通用户用 GUI，极客用 CLI

## 产品哲学

- **你启动我才干活，你关了我就消失。** 不后台常驻、不自启动、不弹窗。
- **宁可漏删，不可误删。** 安全比彻底更重要。
- **越用越聪明。** 不是一次性工具，而是你的磁盘管家。

## 技术架构

```
bakubaku-core（Rust）       — 扫描、分类、记忆、清理引擎
bakubaku-cli（Rust）        — 命令行界面
bakubaku-gui（Tauri+React） — 桌面应用
```

## 技术栈

- **核心引擎：** Rust
- **图形界面：** Tauri 2 + React + TypeScript
- **数据库：** SQLite（记忆系统）
- **大模型：** Claude API / OpenAI API / 本地模型（Ollama）
